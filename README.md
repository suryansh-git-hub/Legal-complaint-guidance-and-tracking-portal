# ⚖️ NyayaPath – Legal Complaint Guidance & Tracking Portal

The platform is used so that if a used for the people who are not able to understand how to deal with legal problem and where to take action for it. The platform helps in guiding them and also provide a way to tell there complaint.

NyayaPath is a full-stack MERN application that enables users to submit legal complaints, upload supporting documents, communicate with administrators, and track the complete complaint resolution process through an intuitive dashboard.

## ✨ Features

### User

- User Registration & Login
- Forgot Password
- Create Legal Complaints
- Upload Supporting Documents
- Track Complaint Status
- Chat with Administrator
- Upload Requested Documents
- View Resolution Details
- Submit Satisfaction Feedback

### Administrator

- Secure Admin Login
- Review Complaints
- Assess Complaints
- Request Additional Documents
- Chat with Users
- Resolve Complaints
- Revise Complaint Resolution
- Update Admin Remarks

## 🔄 Complaint Workflow

User Registration
        ↓
Create Complaint
        ↓
Admin Assessment
        ↓
Request Documents (Optional)
        ↓
User Uploads Documents
        ↓
Conversation Between User & Admin
        ↓
Complaint Resolution
        ↓
User Feedback
        ↓
Resolution Revision (If Required)
        ↓
Complaint Closed
